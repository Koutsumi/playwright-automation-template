# docs/.vitepress

Esta pasta contém a configuração da documentação em `VitePress`.

## O que deve conter

- arquivo de configuração do site;
- customizações de navegação, sidebar e tema quando necessário.

## Regras

- trate esta pasta como infraestrutura da documentação;
- não mova conteúdo de negócio para cá;
- qualquer mudança aqui deve refletir a estrutura real de `docs/`.

## Atenção

Arquivos gerados automaticamente, como cache e build, não devem ser tratados como fonte de verdade nem receber manutenção manual.
